①网站是如何搭建的？

⭐服务器（硬件）：

——PC 电脑，与电脑一样，服务器中某些性能高于办公使用的 PC 电脑，因为其用途不同，所以产生不同名称

——操作系统（并非绑定关系）：家用：windows10、windows8、windows7、windowsXP……

🍓服务器： windows 2000、windows2003、windows2008、windows2012、windows2016、windows2019

——当今服务器类别：独立服务器：——云服务器（VPS），相当于虚拟机

⭐容器(环境)(服务)(中间件)：IIS(windows 自带)、Apache、Tomcat、Nginx……

⭐数据库——大脑 存储数据：Access、SQL server(MSSQL)、MySQL、Oracle

⭐脚本——人：ASP、ASP.NET(ASPX)、PHP、JSP

🍌标配关系（标配关系能达到最大运行性能，容器与脚本非依赖或绑定关系，是默认关系）：

🍓脚本+中间件：

IIS——ASP 或 ASP.NET、ASHX、ASA、CER、CDX

Apache——PHP

Nginx——PHP

Tomcat——JSP

🍓脚本+数据库：

ASP——Access 或 SQL server

ASP.NET(ASPX)——SQL server

PHP——MySQL——中小型企业

JSP——Oracle——大型

②标配关系表：

🍑1、 IIS+ASP+Access——70% 主要用于网站的后台数据存储，负载不高

🍑2、 IIS+ASP+SQL server 主要用于大型网站数据存放

🍑3、 Apache+php+MySQL——90%

🍑4、 Nginx+php+MySQL——90%

🍑5、 Tomcat+jsp+Oracle——20%

🍑6、 Tomcat+jsp+MySQL——80%
