——phpstudy 与 Apache+MySQL+PHP 是什么关系

Phstud 是一个图形化的管理

——phpstudy 中的服务模式与非服务模式

🍉1、启动权限不同

非服务模式——当前登录用户权限（Administrator）

服务模式——以系统权限运行（System）

🍉2、使用方式不同

服务模式及开机后自动启动 Apache+MySQL+PHP

非服务模式及为临时性模式，意味着你注销用户，或重启电脑后，网站将无法继续运行

🍓phpstudy 常识须知：

1、 phpstudy 搭建的 MySQL 数据库中最高权限管理员用户叫做：root （System）密码也为：root

🍓🍓🍓数据库常识须知：

🍓♥每一个数据库不管类型是何种，都会拥有一个账户密码，其中 Access 数据库只拥有密码

各种数据库中的最高权限管理员名称：

1、 MySQL——root

2、 Oracle——sys

3、 SQL server——Sa

MySQL 最高权限管理员账号叫：root，他的密码存储在一个叫 mysql 的数据库，存储在 user 表中，其中 root 账号的密码是默认被加密的

Mysql 数据库管理用户的命名意思：

root@127.0.0.1——只允许当前计算机可以使用该账号连接数据库

root@171.208.14.193——只允许 IP 为 171.208.14.193 的计算机来使用该账户连接 MySQL 数据库
