爱你素素

I love yo

这是我第一篇写博客，可能写的不太好，麻烦见谅。

首先这个写博客的命令，我需要总结一下：

1、更新博客

hexo clean

hexo g -d

一、吐槽
刚开始使用hexo的时候简直无法忍受繁杂的部署命令,不仅要输入以下命令来更新，并且还要在指定的目录下打开 git bash，这让怕麻烦的我十分难受…

hexo clean
hexo g -d

二、搞事情
注意事项：下面的命令中 C:\blog_backup 是我博客的文件夹路径，请替换为你自己博客的路径

任意目录打开 git bash ，输入以下命令

git config –global alias.fuck ‘!cd C:\blog_backup;hexo clean;hexo g -d’
git fuck
以后就可以在任意目录下通过 git fuck 来一键更新了。

三、如何取消别名
git config –global –unset alias.fuck
